# OpsForge

Welcome to **OpsForge**, a consolidated toolkit designed for cybersecurity professionals and cloud infrastructure architects.

## Overview

OpsForge brings together 22 essential utilities into a single, monolithic CLI application, making it easier to audit, monitor, and secure your environments.

### Core Philosophy
- **Deterministic**: No AI-driven guesswork. All tools run predictable logic.
- **Offline First**: Many tools work entirely offline without external API dependencies.
- **Unified Interface**: One command to rule them all.

## Utilities

### Cybersecurity Utilities
- **Port Scanner**: Fast, socket-based scanning.
- **Log Analyzer**: Forensic analysis of server logs.
- **Integrity Monitor**: Cryptographic system file auditing.
- ... and more.

### Cloud Utilities
- **Tag Auditor**: Ensure resource compliance.
- **IAM Linter**: Find permissive security rules.
- **Infra Boilerplate**: Generate standardized Terraform.
- ... and more.

# Cybersecurity Utilities

The **Cyber** module of OpsForge provides a suite of 11 tools designed for offensive and defensive security operations.

## Available Tools

| Tool | Functionality |
| --- | --- |
| [Port Scanner](port-scan.md) | Probes IP addresses for open ports. |
| [Log Analyzer](log-analyze.md) | Parses server logs for malicious patterns. |
| [Integrity Monitor](integrity-check.md) | Audits system file hashes. |
| [Subdomain Enumerator](subdomain-enum.md) | Maps external attack surfaces via DNS. |
| [Password Entropy](password-check.md) | Evaluates password strength offline. |
| [Packet Sniffer](packet-sniff.md) | Captures local network traffic. |
| [Encoder/Decoder](encode-decode.md) | Payload transformation (Base64, Hex, etc). |
| [Directory Discovery](dir-discover.md) | Bruteforces web server directories. |
| [SSL Auditor](ssl-audit.md) | Checks certificate health and crypto. |
| [Honeypot Listener](honeypot.md) | Deceptive service for capturing IP logs. |
| [Vulnerability Fuzzer](fuzzer.md) | Injects test strings into URLs. |
